# Prompt Quizzer – JavaScript Console Quiz Game

This project is a simple quiz game created for WebDev ILab Assignment 4.  
The quiz runs entirely inside the browser console using JavaScript — no HTML or DOM is required.

## Features
- Prompts user with quiz questions using `prompt()`
- Provides immediate feedback with `alert()`
- Uses arrays, loops, conditionals, and functions
- Input normalized using `toLowerCase()` and `trim()`
- Final score displayed at the end

## How to Run
1. Open any web browser (Chrome, Edge, Firefox, etc.)
2. Right-click → Inspect → go to the Console tab
3. Paste the contents of `quiz.js` into the console
4. Press Enter and play the quiz
